# 🔌 Documentação da API (Swagger/OpenAPI)

A nossa camada Backend se auto documenta de forma moderna. O sistema utiliza a geração automática através da biblioteca **FastAPI**.
O padrão Open-Source do mercado (Swagger UI) já está rodando por baixo dos panos.

## Acesso e Teste Interativo
Com o servidor backend ativo na sua máquina (usando o `start.bat`), basta abrir o navegador no endereço oficial da doc:
👉 **[http://localhost:8000/docs](http://localhost:8000/docs)**

Através deste portal, desenvolvedores e QAs podem submeter requisições de teste simuladas antes mesmo de conectar ao React Frontend.

## Catálogo Principal de Endpoints (Restful)

### `/api/ai/summarize` [POST]
- **Objetivo:** Rota vital de ingestão de mídia. Recebe arquivos de áudio/vídeo, envia pro LLM multimodal do Gemini e retorna a sumarização limpa e inteligente.
- **Payload:** `multipart/form-data` contendo a chave `file`.
- **Response Esperado (Code 200):**
```json
{
  "summary": "Texto formatado estritamente em primeira pessoa ocultando CPFs, conforme regras de negócio instruídas ao Agente IA."
}
```

### Endpoints de Autenticação (Novos)

#### `/api/auth/register` [POST]
- **Objetivo:** Criação de novos usuários na plataforma.
- **Regra de Segurança:** Estritamente bloqueado para e-mails que **não** terminem em `@reisrevisional.com.br`.

#### `/api/auth/login` [POST]
- **Objetivo:** Validação de credenciais.
- **Regra de Segurança:** Bloqueia instantaneamente a tentativa de login se o e-mail não for do domínio oficial da empresa.
