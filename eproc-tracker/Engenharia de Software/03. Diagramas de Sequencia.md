# ⏱️ Diagramas de Sequência

Demonstra o fluxo lógico ao longo do tempo para as ações principais, detalhando a comunicação entre os componentes.

## Fluxo 1: Processamento de Áudio por IA e Inserção no DB

```mermaid
sequenceDiagram
    autonumber
    actor CS as Usuário (CS)
    participant Front as Frontend (React)
    participant Back as Backend (FastAPI)
    participant DB as SQLite (Local)
    participant IA as API do Gemini

    CS->>Front: Preenche formulário e anexa Áudio (.mp3)
    CS->>Front: Clica em "Registrar e Processar"
    Front->>Front: Renderiza state "Processando IA..."
    Front->>Back: POST /api/ai/summarize (File via Form-Data)
    
    activate Back
    Back->>IA: client.files.upload(audio)
    activate IA
    Back->>IA: generate_content(file, prompt_padrao)
    IA-->>Back: Retorna String: "Liguei para o cliente e..."
    Back->>IA: client.files.delete() (Limpeza de Segurança)
    deactivate IA
    
    Back->>DB: INSERT INTO attendances (user_id, summary...)
    activate DB
    DB-->>Back: Success / Commited
    deactivate DB
    
    Back-->>Front: { status: 200, "summary": "..." }
    deactivate Back
    
    Front-->>CS: Atualiza Preview na Tela e encerra loading
```
